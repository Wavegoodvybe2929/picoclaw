# Vaults (Obsidian)

Open this `vaults/` folder in Obsidian.

Subfolders:
- `Daily/` — daily logs and summaries
- `Inbox/` — captured tasks/notes
- `Email/` — append-only email logs
- `Calendar/` — append-only calendar change logs
- `Search/` — saved search notes + index
- `Research/` — deep-dive research notes

**Rule:** human-facing notes live in `vaults/`.
